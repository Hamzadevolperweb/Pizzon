# 🍕 Pizzon – Pizza Ordering Website

Pizzon is a modern and responsive pizza ordering website built with **React, Vite, and Tailwind CSS**.
The project focuses on creating a clean UI and smooth user experience for browsing pizza items.

---

## 🚀 Tech Stack

* React.js
* Vite
* TypeScript
* Tailwind CSS
* JavaScript

---

## ✨ Features

* Modern and responsive UI
* Fast performance with Vite
* Clean and structured layout
* Mobile-friendly design
* Easy navigation for food menu

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Hamzadevolperweb/Pizzon.git
```

Go to the project folder:

```
cd Pizzon
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
